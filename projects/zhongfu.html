中复碳芯电缆科技有限公司 商业尽调分析报告（主报告）

报告版本: v1.0（基于尽调材料分析） 分析日期: 2026年6月29日 尽调材料: 中复碳芯企业介绍（2026.3，PPT）、公开行业数据交叉验证（2026.6.29） 分析方法: 纵横分析法 + 多源数据交叉验证 分析框架: 行业 → 纵向演变 → 横向对标 → 估值 → 结论

报告导读

本报告对中复碳芯电缆科技有限公司（以下简称"中复碳芯"或"公司"）进行系统性的商业尽调分析。

核心结论：中复碳芯是中国建材集团旗下碳纤维复合材料制品平台，主营碳纤维复合芯导线（特高压）+风电大梁板（风电）+光伏边框（光伏）三大引擎。公司当前未上市、未申报、未公开任何上市计划（明确判断依据见第1.4节）。2024年主营产品（拉挤大梁板）国内市占率约25%，是国内最早将玻纤大梁板用于风电叶片的企业、目前国内最大的无脱大梁板供应商。注册资本9,346万元，5个生产基地+400余条拉挤大梁板生产线（连云港1#基地105条+2#基地40条UD+北新24条+酒泉103条+张家口86条+中复连众40条），产能5万公里碳纤维芯棒+2万公里导线+10万吨拉挤制品。

5年完整财务数据（企业介绍PPT披露）：

  • 2021年营收3.07亿元/净利3,001万元 → 2025年（未审）营收14.51亿元/净利1.09亿元
  • 2021-2025年营收CAGR：约48%
  • 2021-2025年净利CAGR：约38%
  • 2025年末资产总额12.12亿元/所有者权益3.73亿元/资产负债率69.2%（较高）

投资亮点：央企中国建材集团背书（合计持股约48.54%）+ 碳纤维复合芯导线全球首条1000kV锡林浩特示范工程 + 国内最大无脱大梁板供应商（市占率25%）+ 三引擎协同（特高压+风电+光伏）+ 国家专精特新小巨人 + 三一重能增资入股 + 一带一路20+国家应用。

主要风险：①未上市，无强制披露（2022-2024年审计报告未公开）；②资产负债率较高（69.2%，财务杠杆较大）；③兄弟公司中复神鹰2024年亏损1.24亿元（板块承压）；④光伏边框尚未规模化销售；⑤特高压+风电+光伏三大行业周期叠加波动。

4.3 多估值方法交叉验证

方法1：可比公司PE法

  • 碳纤维赛道平均PE（剔除亏损）：150.4倍（中简+光威主导）
  • 中复碳芯产品定价权弱（净利率4.86%），不应享受龙头PE
  • 按中复碳芯业务结构（碳纤维制品+电缆）综合PE 30-40x × 2025净利润1.09亿 = 33-44亿元

方法2：可比公司PS法

  • 碳纤维赛道平均PS：~5-10x
  • 按中复碳芯2025营收14.51亿 × PS 4-6x = 58-87亿元

方法3：可比公司PB法

  • 行业平均PB：~2-3x
  • 中复碳芯作为央企背景未上市企业，应享受PB溢价
  • 按净资产5亿 × PB 5-8x = 25-40亿元

5.4 最终判断

中复碳芯是央企中国建材集团旗下碳纤维复合材料平台，是国内目前唯一的碳纤维复合芯导线生产商和国内最大的风电大梁板供应商。当前未上市、未申报、未公开任何上市计划（明确判断依据）。新收上限约束：新收上市公司仅 1 家（共 3 平台 = 我方 2 + 新收 1）。两方案对比下的退出方向（爱朋不首选）：

  • 方案 B 选绿田机械：本项目边缘匹配（机械装备业务） → 注入绿田机械，并可通过双主业承接（通用动力机械 + 新材料）扩展碳纤维复合材料主业，可行性中等（机械装备与新材料关联度中等）
  • 方案 C 选科华控股：本项目无法装入科华 → 独立上市（北交所可推演）
  • 扩大新收范围（仅当双主业承接不可行时启动兜底）：电气设备/新材料/复合材料相关上市公司

独立上市路径：北交所（央企背景 + 国家级专精特新）→ 预期 3-5 年 → 核心条件：兄弟公司中复神鹰亏损拖累 + 资产负债率 69.2% → 独立上市路径清晰，但注入绿田机械（方案 B，双主业承接）更优，可缩短退出周期，且免去扩大新收范围带来的新尽调/谈判/控制权风险。

详见 reports/_reserved-projects/listed_companies_injection_analysis.md § 5.7/§ 6/§ 8.6。主要风险：财务不透明、兄弟公司亏损拖累、产品定价权弱、三大行业周期叠加。整体而言，是央企背景的稀缺碳纤维复合材料资产，建议在尽调阶段要求补充完整审计报告、明确上市规划，估值建议50-80亿元区间。
